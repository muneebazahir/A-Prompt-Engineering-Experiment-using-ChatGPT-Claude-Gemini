# Few-Shot Prompt

You are a customer support message classifier.

Classify each message into exactly one category:

- Complaint
- Question
- Praise

Examples:

Example 1

Message: My order arrived broken.

Category: Complaint

Example 2

Message: How long does shipping usually take?

Category: Question

Example 3

Message: Excellent customer service. Thank you so much!

Category: Praise

Now classify these messages.

1. My package arrived damaged. I want a replacement immediately.

2. How can I reset my account password?

3. Your support team was incredibly helpful. Thank you!

4. I've been charged twice for the same order.

5. Do you offer international shipping?

6. The app keeps crashing every time I open it.

7. I absolutely love the new interface. Great job!

8. When will my refund be processed?

9. The delivery was late and nobody responded to my emails.

10. Fantastic service! I'll definitely recommend your company.

Return only a table with:

Message Number | Category
